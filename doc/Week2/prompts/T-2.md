## 内容契约

```python
"""导入所有工具子模块以触发 @tool 注册。"""

from . import add  # noqa: F401

__all__: list[str] = []
```

要点：
- `# noqa: F401` 抑制 ruff 对未使用 import 的告警（这是有意的副作用 import）
- `__all__` 为空 list（无对外符号）
