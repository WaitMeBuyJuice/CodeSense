# B-2: 创建包根 src/codesense_v1/__init__.py

## 任务背景

CodeSense_V1 源码包路径 `src/codesense_v1/`（已由 `pyproject.toml` 声明为 wheel 包根）。需要一个包初始化文件，使 `import codesense_v1` 生效。

---

## 实现目标

创建空的（或仅含版本常量的）包初始化文件，使后续模块（`errors.py` / `schemas.py` / `registry.py` / `server.py` / `tools/`）能以 `from codesense_v1.xxx import ...` 形式被导入。

---

## 需要实现的文件

- `src/codesense_v1/__init__.py`

不得创建/修改任何其他文件（不要创建 `errors.py` 等子模块——属于后续任务）。

---

## 内容契约

文件内容：

```python
__version__ = "0.1.0"
```

要求：
- 仅此一行（或追加 `# -*- coding: utf-8 -*-` 文件头亦可，不强制）
- 不 import 任何子模块
- 不定义 `__all__`

---

## 验收标准

- `uv run python -c "import codesense_v1; print(codesense_v1.__version__)"` 输出 `0.1.0`
- `uv run mypy --strict src/codesense_v1/__init__.py` 零错误
- `uv run ruff check src/codesense_v1/__init__.py` 零警告

---

## 范围约束

- **仅** 创建 `src/codesense_v1/__init__.py`
- 严禁创建 `src/codesense_v1/` 下其他文件
- 严禁修改 `pyproject.toml`
