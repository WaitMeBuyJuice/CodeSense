# 模块理解文档：LLM交互

## 1. 一句话描述

封装大语言模型调用、提示构造与响应处理。

## 2. 对外接口

该模块对外暴露以下接口：

| 符号 | 类型 | 签名 | 说明 |
|---|---|---|---|
| `call_llm` | 函数 | `(prompt: str) -> str` | 接收提示文本，调用 LLM 并返回响应字符串 |

> 注：`src/codesense_v1/llm/__init__.py` 中未导出任何符号，`call_llm` 定义于 `llm.py` 模块内且不以 `_` 开头，符合 Python 公共接口命名惯例。

## 3. 内部文件

| 文件 | 作用说明 |
|---|---|
| `src/codesense_v1/llm/__init__.py` | LLM 子包的初始化文件，当前未导出任何公共符号 |
| `src/codesense_v1/llm/llm.py` | 实现大语言模型调用逻辑，包含 `call_llm` 核心函数 |

## 4. 依赖关系

### 上游依赖（该模块依赖的目录）

- `src/codesense_v1`：作为项目根包，提供包层级上下文与可能共享的初始化配置

### 下游依赖（依赖该模块的目录）

- `src/codesense_v1/summarizer`：依赖 LLM 交互能力生成摘要
- `tests`：包含针对该模块的测试用例
