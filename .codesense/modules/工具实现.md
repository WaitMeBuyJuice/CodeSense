# 模块理解文档：工具实现

## 1. 一句话描述
定义并执行代码分析相关的MCP工具。

## 2. 对外接口
- **`add`** (function): `(a: float, b: float) -> str`
- **`explore_module`** (function): `(module_name: str) -> str`
- **`list_cached`** (function): `() -> str`
- **`list_cached_modules`** (function): `() -> str`
- **`project_map_tool`** (function): `() -> str`

## 3. 内部文件
- `src/codesense_v1/tools/__init__.py`：模块初始化文件，将各个工具组织为一个 Python 包。
- `src/codesense_v1/tools/add.py`：实现基本的加法计算工具。
- `src/codesense_v1/tools/explore_module.py`：实现探索和获取指定模块信息的工具。
- `src/codesense_v1/tools/list_cached.py`：实现列出当前缓存内容的工具。
- `src/codesense_v1/tools/list_cached_modules.py`：实现列出所有已缓存模块列表的工具。
- `src/codesense_v1/tools/project_map_tool.py`：实现生成或获取项目结构映射的工具。

## 4. 依赖关系
- **上游依赖**：
  - `src/codesense_v1`
  - `src/codesense_v1/registry`
  - `src/codesense_v1/summarizer`
- **下游依赖**：
  - `tests`
