# 数据层

## 概述

封装 CodeGraph SQLite 知识图谱的只读查询，提供文件、节点、边的迭代访问，以及目录级依赖聚合与架构特征分析。

## 对外接口

| 符号 | 用途 |
|------|------|
| `CodeGraphDB` | CodeGraph SQLite 数据库的只读封装，支持上下文管理器，提供文件/节点/边的迭代查询 |
| `FileRow` | 文件行数据类，表示一个已索引的文件记录 |
| `NodeRow` | 节点行数据类，表示代码图中的一个符号节点（函数/类/方法等） |
| `EdgeRow` | 边行数据类，表示两个节点之间的依赖关系 |
| `Module` | 表示项目中的一个源文件模块 |
| `ModuleEdge` | 表示两个文件之间的有向依赖边（含外部依赖） |
| `DirCentrality` | 目录级连接度指标（fan-in/fan-out/外部依赖数） |
| `ArchitectureFeatures` | 架构特征聚合包，包含拓扑层级、循环依赖、中心度等信号 |
| `EXTERNAL_PREFIX` | 外部依赖标识前缀常量 |
| `list_modules` | 从 CodeGraph DB 读取所有项目文件为 Module 列表 |
| `module_dependencies` | 计算文件级依赖边集合（支持 import/call） |
| `directory_dependencies` | 将文件级边聚合为目录级依赖字典 |
| `directory_edges` | `directory_dependencies` 的扁平列表变体 |
| `directory_symbols` | 返回目录到符号列表的映射 |
| `compute_centrality` | 计算每个目录的 fan-in/fan-out 中心度 |
| `find_cycles` | 通过 Tarjan SCC 检测目录级循环依赖 |
| `topological_layers` | 将目录按拓扑层级分层（基础层→入口层） |
| `cross_dir_public_api` | 列出每个目录中被外部目录导入的公共符号 |
| `external_dependencies_by_dir` | 按目录聚合外部 import 目标 |
| `architecture_features` | 一次性计算所有语言无关的架构特征信号 |
| `extract_file_docstring` | 从源文件提取模块级文档字符串 |
| `extract_symbol_docstrings` | 批量提取节点的文档字符串 |
| `is_enabled` | 检查文档字符串提取功能是否启用（环境变量控制） |
| `ref_docs_prompt_section` | 生成参考文档列表的 prompt 片段 |
| `_resolve_id` | 用于 import 语句匹配的内部文件标识符 |
| `_file_docstring_for_lang` | 按语言提取文件级文档字符串 |
| `_jsdoc_backward` / `_jsdoc_forward` | JSDoc 注释提取（向前/向后搜索） |
| `_line_comment_backward` / `_line_comment_forward` | 行注释提取（向前/向后搜索） |
| `_python_triple_quote` | Python 三引号文档字符串提取 |

## 内部文件

| 文件 | 作用 |
|------|------|
| `__init__.py` | 模块入口，统一导出数据层公共 API |
| `db.py` | CodeGraph SQLite 数据库只读封装，提供 `CodeGraphDB` 类及 `FileRow`/`NodeRow`/`EdgeRow` 数据类 |
| `modules.py` | 模块级依赖提取：将 DB 中的文件和边转化为 `Module`/`ModuleEdge`，支持内部/外部依赖解析 |
| `aggregate.py` | 将文件级边聚合为目录级视图，提供 `directory_dependencies`、`directory_edges`、`directory_symbols` |
| `architecture.py` | 语言无关的架构特征分析：中心度计算、循环检测、拓扑分层、跨目录公共 API、外部依赖聚合 |
| `files.py` | 文件列表与目录树构建，提供 `list_files` 和 `directory_tree` |
| `docstrings.py` | 多语言文档字符串提取，支持 Python 三引号、JSDoc、行注释等格式 |
| `ref_docs.py` | 参考文档发现，扫描配置的参考文档目录并生成 prompt 片段 |

## 依赖关系

- **上游（该模块依赖）**：无内部依赖（基础层）
- **下游（依赖该模块）**：`src/codesense_v1/summarizer`、`src/codesense_v1/tools`、`scripts`、`tests`
