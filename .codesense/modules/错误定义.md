# 模块理解文档：错误定义

## 1. 一句话描述
定义工具与 LLM 相关的核心错误异常类型。

## 2. 对外接口
该模块对外暴露以下异常类：
- `ToolError`：工具执行过程中发生错误时的基础异常类。
- `ValidationError`：数据或参数验证失败时抛出的异常类。
- `InvalidArgumentError`：接收到无效参数时抛出的异常类。
- `LLMError`：与大型语言模型（LLM）交互时发生错误的异常类。

## 3. 内部文件
- `src/codesense_v1/errors.py`：定义工具执行、参数校验以及 LLM 调用相关的各种错误异常类。
