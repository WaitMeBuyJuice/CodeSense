# 摘要引擎

## 概述

协调数据层（CodeGraphDB）与缓存层（.codesense），驱动 LLM 生成项目级模块划分和各模块 Markdown 摘要，并管理缓存的写入、失效与迁移。

## 对外接口

| 函数 | 签名 | 用途 |
|------|------|------|
| `get_project_map_prompt` | `async (project_root: Path) -> str` | 从 CodeGraphDB 聚合目录结构、符号、依赖、架构层级等信息，构建发送给 LLM 的「项目级模块划分」提示词 |
| `submit_project_map` | `async (project_root: Path, response: str) -> str` | 解析 LLM 返回的管道分隔模块列表，校验目录、展开文件列表、迁移重命名缓存，写入 modules_index 和 project_map.md，返回渲染后的 Markdown |
| `get_module_prompt` | `async (project_root: Path, module_name: str) -> str` | 根据已缓存的模块索引查找指定模块，聚合其文件符号、跨目录公开 API、外部依赖、docstring 等，构建发送给 LLM 的「模块详细摘要」提示词 |
| `save_module_summary` | `(project_root: Path, module_name: str, summary: str) -> None` | 将 LLM 生成的模块摘要写入 `.codesense/modules/<key>.md` 缓存，同时计算并保存模块内容 hash 用于后续失效判断 |

## 内部文件清单

| 文件 | 职责 |
|------|------|
| `src/codesense_v1/summarizer/__init__.py` | 包入口，re-export 四个公开函数：`get_project_map_prompt`、`submit_project_map`、`get_module_prompt`、`save_module_summary` |
| `src/codesense_v1/summarizer/summarizer.py` | 核心实现。包含：① 公开 API（4 个 async/sync 函数）；② 模块文本解析与目录校验（`_parse_modules_text`、`_normalize_dir`）；③ 文件列表展开与去重（`_expand_module_files`）；④ Markdown 渲染（`_render_project_map_markdown`）；⑤ 提示词构建（`_build_project_map_prompt`、`_build_module_prompt`）；⑥ 缓存 hash 计算与重命名迁移（`_compute_module_hash`、`_migrate_renamed_module_caches`）；⑦ 辅助工具函数（目录分类、根目录解析、语言推断等） |

## 依赖关系

### 上游（本模块依赖）

| 模块 | 使用方式 |
|------|----------|
| `codesense_v1.data` | 从 CodeGraphDB 读取模块列表、依赖边、目录符号、公开 API、外部依赖、docstring 等 |
| `codesense_v1.cache` | 读写 `.codesense` 下的 modules_index、project_map、module .md 缓存及 hash 文件 |
| `codesense_v1.errors` | 抛出 `InvalidArgumentError` |

### 下游（依赖本模块）

| 模块 | 使用方式 |
|------|----------|
| `codesense_v1.tools` | MCP 工具层调用 `get_project_map_prompt`、`submit_project_map`、`get_module_prompt`、`save_module_summary` |
| `tests` | 单元测试覆盖解析、展开、渲染、缓存迁移等逻辑 |

## 外部依赖库

| 库 | 用途 |
|----|------|
| `difflib` | 目录名模糊匹配（`_normalize_dir`，cutoff=0.85） |
| `hashlib` | 模块内容 SHA-1 hash 计算（`_compute_module_hash`） |
| `os` | 读取环境变量 `CODESENSE_INCLUDE_DIRS`、`CODESENSE_CACHE_AUTO_EXPIRE` |
| `pathlib` | 路径操作（`Path`） |
| `re` | 正则：文件扩展名检测、辅助目录分词、描述去重分割 |
