# 模块理解文档：注册中心

## 1. 一句话描述
统一注册、发现并调度分发工具等服务能力。

## 2. 对外接口
根据 Python 命名惯例（不以 `_` 开头的符号视为公开接口），该模块对外暴露以下接口：

* **`ToolSpec`** (class): 工具规范定义类，用于结构化描述工具的元信息。
* **`tool`** (function): 工具注册装饰器，接收工具名称、描述和输入 Schema，返回对应的装饰器函数。
* **`deco`** (function): 实际的装饰器处理函数，用于将目标函数包装并注册为 ToolHandler。
* **`list_tools`** (function): 获取并返回当前已注册的所有工具列表。
* **`dispatch`** (function): 工具调度分发器，根据传入的工具名称和参数执行对应的工具并返回结果。

## 3. 内部文件
* **`src/codesense_v1/registry/__init__.py`**: 模块包初始化文件，用于导出 registry 中的核心接口供外部调用。
* **`src/codesense_v1/registry/registry.py`**: 实现工具注册、发现、参数校验及调用分发的核心业务逻辑。
