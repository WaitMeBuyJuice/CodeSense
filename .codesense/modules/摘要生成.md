# 模块理解文档：摘要生成

## 1. 一句话描述
对代码库或代码实体生成结构化摘要。

## 2. 对外接口
根据 Python 命名惯例，不以 `_` 开头的符号为对外暴露的公开接口：
- **`project_map_summary`**: `(project_root: Path) -> str`
  生成并返回整个项目结构的摘要。
- **`module_summary`**: `(project_root: Path, module_name: str) -> str`
  根据项目路径和模块名生成并返回特定模块的摘要。

## 3. 内部文件
- `src/codesense_v1/summarizer/__init__.py`: 模块的初始化文件，用于将包目录标记为 Python 模块。
- `src/codesense_v1/summarizer/summarizer.py`: 实现摘要生成的核心逻辑，包含目录过滤、依赖收集、大模型(LLM)提示词构建与调用、以及最终 Markdown 格式渲染等辅助函数。

## 4. 依赖关系
- **上游（该模块依赖的目录）**:
  - `src/codesense_v1`
  - `src/codesense_v1/cache`
  - `src/codesense_v1/data`
  - `src/codesense_v1/llm`
- **下游（依赖该模块的目录）**:
  - `src/codesense_v1/resources`
  - `src/codesense_v1/tools`
